## Tabela ASCII

Combinar com ALT + Número:

30 = ▲ (seta pra cima)

31 = ▼ (seta para baixo)

16 = ► (seta para direita)

17 = ◄ (seta para esquerda)

Outro formato:

23 = ↨ (seta para baixo e para cima)

24 = ↑ (seta pra cima)

25 = ↓ (seta para baixo)

26 = → (seta pra direita)

27 = ← (seta para esquerda)
